## 🔹 AI Legal Document Generator (RAG + LLM)

### 🧩 Problem Statement

Manual legal document creation is time-consuming, inconsistent, and heavily dependent on domain experts.

### 💡 Solution

Designed a Generative AI solution using LLMs with Retrieval-Augmented Generation (RAG) to automate structured document creation.

### ⚙️ Tech Stack

LLMs (OpenAI/Claude), LangChain, Vector DB (FAISS/Pinecone), Python, Flask

### 🎯 Use Cases

* Contract generation based on user inputs
* Dynamic modification of legal agreements

### 🔄 System Flow

User Input → API → RAG Pipeline → LLM → Structured Document Output

### 📈 Outcome

* Reduced document creation time
* Improved consistency and accuracy
* Scalable across multiple document types
